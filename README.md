<h2 align="center">
  NAME CREATOR MANAGER
</h2>

Hello there!

This was a quick project I did to help me manage a lot of names for my other project [Moba Manager](https://github.com/08darknight1/MobaManager)!

It is a simple JAVA application, letting you add first names, nicknames and surnames to an excel file stored in the project.

After you are done adding your desired names, you can export the file as a CSV to be used somewhere else.

There are still some features missing like being able to add/edit the columns, see a full list of the names already added and etc.

These will stay in the drawing board for now as to not hinder the development of Moba Manager!
